INTRODUCTION.

The angularjs patterns are borrowed from around the web and heavily from projects like mean.js and mean.io. This project basically represents hours of research and planing to create a system that moves quickly through development and is flexible for many projects and many purposes. Package management is not limited I have tried to leave as many avenues for package configuration open, while also providing the interface for a more strict package management system, at the developer's discretion. The site in its native state makes use of bower, and composer for various features however neither system is required for deployment.
