Welcome

This framework has been constructed from the top down by Matt Baker it combines MongoDB AngularJS and PHP. Its goal is to couple a backend platform on open source technologies with a modern client driven front end, while maintaining a flexible and easy to use architecture. The framework consists of three components:
1). A stateless RESTFUL API with dynamic routing and token authentication
written in PHP with minimal code to power operations on the data layer (future versions to support MYSQL )
2). A simple feature-rich aggregation component
written in PHP to power the view layer, implement code optimization, and provide access to front-end libraries.
3). An AngularJs front end MVC for powerful client-driven 'single page app' behavior
